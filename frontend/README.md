# Frontend CreviPlay

Frontend básico para el proyecto CreviPlay - Estudio de Desarrollo de Videojuegos.

## Estructura

```
frontend/
├── index.html      # Página principal con todas las secciones
├── css/
│   └── styles.css  # Estilos CSS
├── js/
│   └── script.js   # JavaScript para funcionalidad
└── README.md       # Este archivo
```

## Características

- **Diseño responsive**: Funciona en dispositivos móviles y desktop
- **Navegación suave**: Transiciones entre secciones
- **Secciones incluidas**:
  - Inicio (Hero section)
  - Portafolio de videojuegos
  - Sobre nosotros
  - Formulario de contacto

## Uso

1. Abre `index.html` en un navegador web
2. Navega entre las secciones usando el menú superior
3. El formulario de contacto está funcional (frontend básico)

## Notas

- Este es un frontend básico para demostración
- En producción, el formulario de contacto debería conectarse a un backend
- Las imágenes de los proyectos son placeholders
- Diseño simple y funcional, apropiado para proyecto de 2º ASIR

## Personalización

- Colores: Edita las variables de color en `css/styles.css`
- Contenido: Modifica el HTML en `index.html`
- Funcionalidad: Añade más JavaScript en `js/script.js`

## Integración con WordPress

Este frontend puede servir como referencia para el diseño del tema WordPress que se implementará en el proyecto final.
